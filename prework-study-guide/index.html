    Git

  • git status: checks what branch we are currently on.
  • git checkout -b "branch name": creates a new branch and switches to it.
  • git pull origin main: Information from the main branch is pulled into your current branch.
  • git push "branch name": Information from your current branch is pushed toward the main branch.
  • git add -A: Adds modifications in your current branch to the staging area.
  • git commit -m "description": adds a description to the current files in the staging area.
